--------

1. DETECT DISEASE API
---------------------

API URL:
/api/detect-disease

Full URL:
http://localhost:3000/api/detect-disease

API Type:
POST

Headers:
Content-Type: application/json

Body Parameters:

Parameter: text
Type: string
Required: Yes
Description: User symptom text. Supports English, Hindi, Hinglish, Arabic, Bhojpuri, and mixed language text.

Request Body Example:
{
  "text": "मुझे सिर दर्द और उल्टी जैसा लग रहा है"
}

POSTMAN TESTING STEPS
---------------------

Detect Disease:
1. Open Postman.
2. Select POST method.
3. Enter URL: http://localhost:3000/api/detect-disease
4. Go to Headers and add:
   Content-Type: application/json
5. Go to Body.
6. Select raw.
7. Select JSON.
8. Add:
   {
     "text": "I have fever and headache"
   }
9. Click Send.

Default Reminders:
1. Open Postman.
2. Select GET method.
3. Enter URL: http://localhost:3000/api/reminders/defaults
4. Click Send.


IMPORTANT MEDICAL DISCLAIMER
----------------------------
This API does not provide a confirmed medical diagnosis.
It only returns possible conditions based on symptoms.
Users should consult a doctor or healthcare professional for proper diagnosis and treatment.
